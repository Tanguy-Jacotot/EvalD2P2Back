EVALD2P2
Contexte :
Vous travaillez pour une entreprise qui organise régulièrement des événements professionnels. 
L'entreprise souhaite digitaliser la gestion de ces événements à travers une application web pour faciliter leur organisation 
et la participation des utilisateurs.


Objectif
Développer une application web en utilisant .NET pour le back-end avec des Azure Functions et Angular pour le front-end. 
L'application permettra aux utilisateurs de créer, lister, modifier et supprimer des événements. 
Chaque événement contiendra des informations telles que le titre, la description, la date, l'heure et le lieu.


Prérequis
Visual Studio 2022 / JetBrains Rider 2022.3.2
.NET 7.0


Installation
Cloner le projet depuis la branche main sur le dépôt Git
Ouvrir le projet dans Visual Studio ou Rider
Lancer le projet (en HTTPS ou en HTTP)
Ouvrir le navigateur à l'adresse suivante : https://localhost:7207 ou http://localhost:5130
Vous pouvez maintenant naviguer sur le site
